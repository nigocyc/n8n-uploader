Privacy Policy

Last updated: June 7, 2026

This Privacy Policy describes how n8n-uploader ("the application") handles information when it accesses Google APIs. The application is a personal-use automation tool used to upload video content to the authenticated user's own YouTube channel.

Information we access

The application requests access to the following Google API scope:

  • https://www.googleapis.com/auth/youtube.upload — used solely to upload finished video files to the authenticated user's YouTube channel.

The application does not request access to read, edit, or delete existing YouTube videos, comments, playlists, analytics, or any other Google account data.

How we use the information

Access to the YouTube account is used only to perform video uploads initiated by the user's own automated workflow. The application does not display, share, or transfer this access or any associated data to any third party.

Data storage

The OAuth credentials (access and refresh tokens) generated during authorization are stored locally on the user's own server and are used only to authenticate upload requests to YouTube. No user data is stored on any external database or transmitted to any server other than Google's official API endpoints.

Limited Use disclosure

n8n-uploader's use and transfer of information received from Google APIs to any other app will adhere to the Google API Services User Data Policy, including the Limited Use requirements.

Data sharing

We do not sell, rent, or share any data accessed through Google APIs with any third party. All access is used exclusively for the application's stated upload function.

Data retention and deletion

Stored OAuth tokens can be revoked at any time by the user through their Google Account permissions page. Revoking access immediately and permanently disables the application's ability to upload on the user's behalf.
