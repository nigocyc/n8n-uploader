n8n-uploader

A personal automation tool that publishes short-form video content to my own YouTube channel.

What this application does

n8n-uploader is a personal-use automation pipeline I built to publish educational short videos to my own YouTube channel. Each video is generated automatically from a content schedule, narrated with text-to-speech, illustrated, and assembled into a final clip. Once a video is ready, the application uploads it to YouTube on my behalf.

How it works

1. Plan

Topics are scheduled in a content table.

2. Generate

Script, voiceover, images, and video are created automatically.

3. Publish

The finished video is uploaded to YouTube via the YouTube Data API.

Google account access

This application requests the youtube.upload scope for one purpose only: to upload finished video files to the authenticated user's own YouTube channel. It does not read, modify, or delete any other YouTube data, and it does not access any other Google services.
